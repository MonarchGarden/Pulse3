# Pulse3

Pulse3 is a read-only Web3 gas intelligence mobile app built with Flutter.

## Features

- Multi-chain gas tracking (Ethereum, Arbitrum, Base)
- Auto-refresh gas polling
- Gas trend indicators (↑ ↓)
- Clean, mobile-first UI
- No wallet, no signing, no custody

## Tech Stack

- Flutter
- Riverpod
- Dio
- Public RPC endpoints

## Philosophy

Pulse3 is designed to be:

- Safe (read-only)
- Lightweight
- Mobile-first
- Low-maintenance
